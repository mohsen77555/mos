# 🌾 تطبيق صيانة مطحنة الدقيق — Flour Mill Maintenance App

تطبيق ويب تقدمي (PWA) بواجهة عربية كاملة (من اليمين لليسار) لإدارة صيانة مطحنة الدقيق.
يعمل على الهاتف والحاسوب، **دون اتصال بالإنترنت**، ويحفظ كل البيانات **محلياً على جهازك**.

## ✨ المميزات

| الوحدة | الوصف |
|--------|-------|
| ⚙️ **سجل المعدات** | تسجيل المطاحن والمناخل والمحركات والسيور… مع النوع والموقع والرقم التسلسلي والحالة (تعمل / متوقفة / تحت الصيانة). |
| 🗓️ **جدول الصيانة** | جدولة الصيانة الدورية (تشحيم، تنظيف، فحص…) مع تكرار بالأيام وحساب تلقائي للموعد القادم وتنبيهات الاستحقاق والتأخير. |
| 🔧 **أوامر العمل** | تسجيل الأعطال والإصلاحات مع الأولوية والفني والقطع المستخدمة والتكلفة والحالة. |
| 📦 **قطع الغيار** | إدارة المخزون مع حد أدنى وتنبيه «نفاد قريب» وأزرار سريعة لزيادة/إنقاص الكمية. |
| 🏠 **لوحة التحكم** | ملخص فوري: عدد المعدات، المتوقفة، أوامر العمل المفتوحة، الصيانة المستحقة، والمخزون المنخفض. |
| ⇅ **نسخ احتياطي** | تصدير/استيراد كل البيانات بصيغة JSON. |

## 🚀 كيفية التشغيل

التطبيق ملفات ثابتة فقط — لا يحتاج خادماً ولا تثبيت حزم.

### الطريقة الأسرع (محلياً)
شغّل خادماً بسيطاً داخل المجلد ثم افتح المتصفح:

```bash
# باستخدام بايثون
python3 -m http.server 8000
# ثم افتح:  http://localhost:8000
```

> ملاحظة: عامل الخدمة (العمل دون اتصال) يحتاج إلى `http://localhost` أو موقع `https`.
> فتح `index.html` مباشرة بنقرة مزدوجة يعمل لكن دون ميزة «دون اتصال».

### التثبيت على الهاتف (Add to Home Screen)
1. افتح رابط التطبيق في متصفح الهاتف (Chrome / Safari).
2. من قائمة المتصفح اختر **«إضافة إلى الشاشة الرئيسية»**.
3. سيظهر التطبيق كأيقونة مستقلة ويعمل دون اتصال.

### النشر المجاني (اختياري)
ارفع الملفات إلى أي استضافة ثابتة مثل **GitHub Pages** أو **Netlify** أو **Vercel** للوصول إليه من أي مكان.

## 🗂️ هيكل المشروع

```
.
├── index.html              # الصفحة الرئيسية والهيكل
├── manifest.webmanifest    # إعدادات PWA (الاسم، الأيقونة، الألوان)
├── sw.js                   # عامل الخدمة للعمل دون اتصال
├── css/styles.css          # التنسيقات (واجهة RTL محمولة)
├── js/app.js               # منطق التطبيق وقاعدة البيانات المحلية
└── icons/icon.svg          # أيقونة التطبيق
```

## 💾 عن البيانات

- تُخزَّن جميع البيانات في **`localStorage`** داخل المتصفح على جهازك فقط.
- لا تُرسَل أي بيانات إلى أي خادم.
- استخدم زر **⇅** في الأعلى لعمل **نسخة احتياطية** دورية (تصدير ملف JSON) خصوصاً قبل تغيير الجهاز أو المتصفح.
